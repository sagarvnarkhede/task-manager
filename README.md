# Task Manager Installation
###### Note: If you restart the server application will clean the existing data,  next time user has to add create new task list

#### Node Version 
* v10.16.0
    * nvm i v10.16.0
    * nvm use v10.16.0

### Stat Rest Server 
 * task-rest
    * cd task-rest
    * npm i
    * node app.js

### Start UI

 * task-app
    * cd task-app
    * npm i
    * npm run serve
